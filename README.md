An [UserScripts](https://www.userscript.zone/) for [Tampermonkey](https://github.com/Tampermonkey/tampermonkey) or [Violentmonkey](https://github.com/violentmonkey/violentmonkey) extension on [Firefox](https://www.firefox.com/) Desktop/Mobile and [Chrome](https://www.google.com/chrome/) Desktop, and may be work for other script extension, and also for Mobile Browser based on [Chromium](https://github.com/chromium/chromium) with native user.js support like [Cromite](https://github.com/uazo/cromite) and [Via Browser](https://github.com/tuyafeng/Via). All this userscript also published at [Greasyfork](https://greasyfork.org/en/users/1326264-mochamad-adi), kindly give a good rating if you were helped.
