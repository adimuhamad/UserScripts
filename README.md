An [UserScripts](https://www.userscript.zone/) for [Tampermonkey](https://github.com/Tampermonkey/tampermonkey) or [Violentmonkey](https://github.com/violentmonkey/violentmonkey) extension on [Firefox](https://www.firefox.com/) Desktop/Mobile and [Chrome](https://www.google.com/chrome/) Desktop, and may work for other script extension too, and also for [Chromium](https://github.com/chromium/chromium)-based Mobile browsers with native user.js support such as [Cromite](https://github.com/uazo/cromite) and [Via Browser](https://github.com/tuyafeng/Via). All these userscript also published on [Greasyfork](https://greasyfork.org/en/users/1326264-mochamad-adi), kindly give a good rating if you were helped.
